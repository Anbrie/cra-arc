Creating security questions and answers

Creating security questions and answers is a security feature that is part of our ongoing commitment to bring you secure login services and to protect your personal information.

For logins, you may have to answer one of the security questions you created to validate your identity. This may happen if you are trying to access our services from an unrecognized computer.

It is important that you choose questions and answers that you can easily remember but are hard for others to guess. Your security questions and answers are as important as your personal information. Do not disclose them to anyone.

From each drop-down list, select one question to answer. At the end of this process, you should have five questions and five written answers.

Additional security feature preference

If you select "Ask me a security question each time I login using this device," each time you login you will be asked to provide the answer to one of the questions that you chose.

If this is a trusted device and you choose "Do not ask me a security question each time I login using this device," a persistent cookie will be added to your browser. As a result, when you login you will not have to answer any of the questions that you chose.

Note

We recommend that you select "Do not ask me a security question each time I login using this device" only for your personal device and not one that is shared by others - for example, in Internet cafés or public libraries.
