Content approach

Purpose of equal height

Use to make grid cells the same height. Make the inner elements (panels, wells and so forth) inherit the height from the parent grid.

Correct use

  • The class hght-inhrt is applied on the element to be equalize as well as all the parents (up to the child of wb-eqht), whereas eqht-trgt is only applied to the element to equalize (nothing needs to be done about the parents).
