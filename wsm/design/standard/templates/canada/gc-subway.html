How to implement

  1. Create a main page

    [details]

  2. Create first level sub-pages

    You have three choices, depending on whether or not this first level page has a second level of IA.

    No second level of IA from page

    [details]

    Page has a second level of IA

    [details]

    If your product has several first level pages that have a secondary IA, you can choose the option that best suites that page. You do not have to stick to only one option within the same product.

    Option 1: First level page is a navigation-only page

    [details]

    Option 2: First level page is a content page that has link to second level within the content flow

    [details]

  3. If needed, create second level sub-pages

    You have three choices, depending on whether or not there is a sequence the user should follow on the second level of IA versus if these pages are standalone pages that are not part of a flow or process.

    Second level has sequence

    [details]

    Second level has no sequence

    [details]

    Option 1: Show all items

    [details]

    Option 2: Show only the active item

    [details]
