Content approach

Purpose

Labels provide additional context about the adjacent content. Labels call attention to a section of content by adding short, text-based bits of supporting information.

Design options

Basic labels

Use the appropriate label to describe and highlight text-based (non-numerical) information.

Correct use

General
  • Use to feature a few words within a coloured background
  • Do not use in alerts or buttons
  • Do not use to highlight unread items − use Badges
  • Do not highlight large sections of text − use Alerts
Info label
  • Use .label-info to indicate that the content is informative in nature
Primary label
  • Use .label-primary to match the label to the colour theme of the site
Success label
  • Use .label-success to indicate that the content is correct
Warning label
  • Use .label-warning to indicate that the user needs to be warned
Danger label
  • Use .label-danger to indicate that the content or action is dangerous
