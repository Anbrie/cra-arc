Creating a user centred design - CRA web service manual

The CRA’s User Centered Design process is essential to designing a website that is tailored to meet the needs of Canadians using Canada.ca. This is done by connecting the UX researchers and designers with users, which is critical to developing empathy for those who use our web services. The purpose is to produce web content that is easy to find and understand and is needed for users to complete their goals.

The process involves a wide range of activities including user research, User Interface (UI) design, usability testing, and creating an intuitive Information Architecture (IA).
