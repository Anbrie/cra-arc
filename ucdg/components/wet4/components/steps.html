Correct use

Numbered lists
  • Use ordered lists to show:
    • ranking
    • order
    • priority
    • step-by-step instructions
Alpha lists
  • Use to display letters instead of numbers
    • This applies only to immediate children list items
    • This means you also need to add the CSS class for any nested lists
  • Use lower alpha mainly as a nested second level list
  • Use upper alpha mainly in legal documents
Roman numeral
  • Use to display Roman (I, II, III, IV or i, ii, iii, iv…) instead of Arabic (1, 2, 3, 4…) numerals
    • This applies only to immediate children list items
    • This means you also need to add the CSS class for any nested lists
  • Use lower roman mainly as a nested second level list
  • Use upper roman mainly in legal documents

Stylized steps

Use to outline the process to complete a task that can be divided into steps and sub-steps.

Headings

  • Using section/div:
    • With headings

Correct use

  • List items in the correct order for task completion
  • Start with an active verb (for example, open, complete, manage, know, close, send, keep, find, etc.)
  • If sub steps do not have to be in a specific order, bullet points are acceptable

Incorrect use

  • Do not use if the order of the list items does not explicitly matter
  • Do not exceed 4 levels of sub steps

Complementary components

Additional add-on features and behaviours are available.
