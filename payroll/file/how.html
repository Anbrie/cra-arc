Which method to use

You should file all returns electronically to avoid the delays that may occur in processing paper returns. If you have more than one payroll account, you have to file a separate return for each account. You may file information returns online or on paper. However, you must file returns online if you file more than 50 slips for a calendar year.

Web forms and internet file transfer

  • Web Forms

    Web Forms allows you to complete an information return easily, following step-by-step instructions.

    Benefits to using Web Forms
    • You can file up to 100 slips (original, additional, amended, or cancelled) from our website
    • Create an electronic information return containing slips and a summary, which can be saved and imported at a later date
    • Summary totals are calculated for you
    • Allows you to print your slips and your summary
    • The application validates data in real time

  • Internet file transfer

    Internet file transfer allows you to use payroll, commercial, or in-house developed software, to submit an xml file of up to 150 MB over the Internet.

Learn more on how to file returns electronically: Filing Information Returns Electronically (T4/T5 and other types of returns).

We do not accept returns filed on magnetic tapes, CDs, DVDs, USB keys or diskettes.

Do not send a paper copy of the slips or summary.

On paper – 50 slips or less

We encourage you to file your return electronically. However, if you have 50 slips or less, you can file on paper.

If you are filing slips on paper:

  • You must fill out one copy of each slip you need to fill out for each employee and include it with your summary when you file.
  • Enter the information for two different employees on one sheet.
  • Mail the return to the address provided on the summary.
  • Keep a copy of the slips and summary in your records.

What to do if you are sending a large quantity

We accept alternate versions of the slip if you are sending a large quantity. For more information, see Customized forms.

What happens if you file a summary without slips

If a summary is sent without the slips, the following applies:

  • we will consider the return as not received
  • the summary will not be returned
  • a complete return must be filed including the slips

What happens if you file using the wrong method

If you file more than 50 information returns for a calendar year, you must file the returns electronically. After filing the original return, slips can be amended added or deleted in either paper or electronic format.

If you do not file electronically when you are required to do so, the penalty we assess is based on the number of information returns filed in an incorrect format. Each slip is an information return.

The penalty is calculated per type of information return. For example, if you file 51 NR4 slips and 51 T4 slips on paper, we would assess two penalties of $250, one per type of information return.
