Before you start

To use Auto-fill my return (AFR), you must first use the current year’s version of the UFile CVITP software to submit the authorization request, then follow the step-by-step instructions to download the individual’s information.

Remember the following:

  • Close all web browsers that you have running before you attempt to download an individual’s information through AFR
  • Have your CRA user ID and password or sign-in partner information available. This is the same information you would use to access other CRA login services, such as Represent a Client, My Account, and My Business Account. You will need these later in the process
  • Set the time and time zone correctly on your computer

Spousal information

On this screen

If you are providing spousal information, you must do so before you submit an Authorization request for the Family Head.

  • Click add spouse
  • Complete the Identification and CRA questions sections for the spouse
  • Return to the Family Head’s profile in the navigation bar and proceed with the AFR process

If you are also using AFR to file the spouse’s return, you must also submit an Authorization request for the spouse. This can be done at any time once you have completed the Identification and CRA questions for the spouse.

Authorization request  – signature page

On this screen

  • On the Tax Return tab, select CRA Authorization request from the left side menu to populate the Authorization request – signature page from the left side menu
  • Print the Authorization request – signature page
  • Read the script to the individual
  • Ask the individual to sign the page

Only the individual keeps a copy of this page. As a volunteers, you do not retain paper copies.

Authorization request  – signature page in a virtual setting

  • On the Tax Return tab, select CRA Authorization request from the left side menu to populate the Authorization request – signature page from the left side menu
  • Read the script for virtual situations to the individual
  • Submit the authorization request once verbal permission has been received

Accessing CRA web services (AFR)

On this page
Your web browser will open the CRA's Tax information web service page.
Use one of the two methods to access the AFR application:
1.     If you are using the CRA login method, enter your CRA user ID and password
2.     If you are using the Sign-In Partner Login method, login using your online banking information

This is the same information you would use to access other CRA login services, such as Represent a Client, My Account, and My Business Account. If you run into issues with Multi-factor authentication, see Multi-factor authentication – Auto-fill my return (AFR) and CVITP volunteers.

AFR - Verify the SIN

On this screen
Once you have logged in, the system will take you to the Tax information web service-request confirmation screen and ask you to verify the SIN you are using.

With the UFile CVITP software, you can only enter one SIN at a time.

  • Once you confirm that the SIN is correct, click Next 
  • You will need to wait a few minutes for the system to process your request

Electronic filing – EFILE
On this screen

 Although the authorization request is transmitted through the current year UFile software, you must still answer Yes to this question in 2017 UFile.

  •  In the left side menu, select EFILE
  • Select Yes to the question that asks you if you will be filing an authorization request to use the CRA Auto-fill my return service
  • Click Next

This will bring you to the T4 and employment income page.

Form T1013 was replaced by the Authorization request – signature page beginning in the 2019 tax filing season. Any reference to Form T1013, Authorizing or Cancelling a Representative in the 2017 software is the same as a reference to the Authorization request – signature page in 2019 and subsequent versions of the software.
