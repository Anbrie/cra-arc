Content approach

Purpose of footnotes

The purpose of the Footnotes plugin is to implement a consistent, accessible way of handling footnotes across Government of Canada web sites. The main concept behind this solution is to place footnotes in a definition list, within a dedicated section. Supporting CSS is used to lay out the footnotes and hide navigational aids.
