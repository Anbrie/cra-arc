CRA sign-in services

Add multi-factor authentication to your account

The CRA is gradually adding multi-factor authentication for all users which will help make your CRA sign-in services more secure! You can now add multi-factor authentication to your CRA sign-in services. Add this feature by signing in and following these instructions. Once you add this feature, it will be automatically added to all your CRA sign-in services.

If your CRA user ID and password have been revoked

Some taxpayers may have received a notification that their CRA user ID and password have been revoked. Visit CRA user ID and password have been revoked for more information.
